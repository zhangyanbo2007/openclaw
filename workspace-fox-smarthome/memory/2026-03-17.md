## 🎯 经验总结

1. **猫眼告警延迟**: 从检测到人到告警产生约需 25-30 秒
2. **查询时机**: 必须等待至少 30 秒再查询告警
3. **告警图片**: 使用 `alarmPicUrl`（告警时的照片），不是实时抓图
4. **API 格式**: 萤石 API 返回的 `data` 直接是数组，不是 `.data.list`
5. **进门优先**: 有告警=进门，无告警=出门
6. **阿里云人脸识别**: SearchFace API 只支持 HTTP/HTTPS URL，不支持 Base64
7. **事件触发器调用脚本**: 确保调用 `door_handler_v2.sh`（不是旧版 `door_handler.sh`）
