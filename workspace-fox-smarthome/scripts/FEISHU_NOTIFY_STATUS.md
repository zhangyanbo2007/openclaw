## 🔧 配置选项

### 选项 1：使用系统消息工具（推荐）
修改 `door_handler_v2.sh`，直接调用 `openclaw message` 工具：

```bash
# 进门通知
openclaw message send --channel=feishu \
  --target=ou_0671c63ec5781695e9ca1ec20c78f94e \
  --message="🏠 小王子回家了！..."

# 出门通知
openclaw message send --channel=feishu \
  --target=ou_0671c63ec5781695e9ca1ec20c78f94e \
  --message="👋 小王子出门了..."
```

**优点**：
- ✅ 不需要应用凭证
- ✅ 使用用户身份发送
- ✅ 更安全可靠

**缺点**：
- ⚠️ 需要用户已授权飞书

---

### 选项 2：配置飞书应用凭证
在 OpenClaw 配置中添加飞书应用信息：

```yaml
feishu:
  appId: "cli_xxx"
  appSecret: "xxx"
```

**优点**：
- ✅ 脚本自动发送
- ✅ 不需要用户干预

**缺点**：
- ⚠️ 需要应用级权限
- ⚠️ 配置复杂
